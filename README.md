Ниже приведены открытые наборы данных с временными рядами, которые можно использовать для анализа и моделирования:
---

### **1\. Финансовые данные (цены акций, валютные курсы)**

#### **Yahoo Finance**

* **Описание**: Исторические данные о ценах акций, индексах, валютах и других финансовых инструментах.  
* **Ссылка**: [Yahoo Finance](https://finance.yahoo.com/)  
* **Как скачать**: Используйте API или вручную экспортируйте данные в CSV.

#### **Alpha Vantage**

* **Описание**: Бесплатный API для получения исторических данных о ценах акций, криптовалют, валютных курсов и других финансовых инструментов.  
* **Ссылка**: [Alpha Vantage](https://www.alphavantage.co/)  
* **Как скачать**: Зарегистрируйтесь для получения API-ключа и используйте его для загрузки данных.

---

### **2\. Погодные данные**

#### **NOAA (National Oceanic and Atmospheric Administration)**

* **Описание**: Данные о погоде, включая температуру, осадки, влажность и другие метеорологические параметры.  
* **Ссылка**: [NOAA Climate Data](https://www.ncdc.noaa.gov/)  
* **Как скачать**: Используйте инструменты поиска на сайте для загрузки данных в формате CSV.

#### **OpenWeatherMap**

* **Описание**: Исторические и текущие данные о погоде по всему миру.  
* **Ссылка**: [OpenWeatherMap](https://openweathermap.org/api)  
* **Как скачать**: Зарегистрируйтесь для получения API-ключа и используйте его для загрузки данных.

---

### **3\. Транспортные данные (загруженность дорог)**

#### **Uber Movement**

* **Описание**: Данные о времени поездок и загруженности дорог в различных городах мира.  
* **Ссылка**: [Uber Movement](https://movement.uber.com/)  
* **Как скачать**: Зарегистрируйтесь и скачайте данные в формате CSV.

#### **Google Maps API**

* **Описание**: Данные о загруженности дорог и времени в пути.  
* **Ссылка**: [Google Maps API](https://developers.google.com/maps/documentation)  
* **Как скачать**: Зарегистрируйтесь для получения API-ключа и используйте его для загрузки данных.

---

### **4\. Потребление ресурсов (вода, электроэнергия)**

#### **PJM Interconnection**

* **Описание**: Данные о потреблении электроэнергии в реальном времени и исторические данные.  
* **Ссылка**: [PJM Data Miner 2](https://dataminer2.pjm.com/)  
* **Как скачать**: Используйте инструменты на сайте для загрузки данных в формате CSV.

#### **Water Data for the Nation (USGS)**

* **Описание**: Данные о потреблении воды, уровне рек и других гидрологических параметрах.  
* **Ссылка**: [USGS Water Data](https://waterdata.usgs.gov/nwis)  
* **Как скачать**: Используйте инструменты поиска на сайте для загрузки данных.

---
